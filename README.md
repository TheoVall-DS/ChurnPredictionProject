# Прогнозирование оттока клиентов интернет-магазина
Финальный проект специализации МФТИ и Яндекса "Машинное обучение и анализ данных"
### Описание проекта
Имеются данные об оттоке пользователей за некоторый период. При этом о переменных ничего неизвестно, кроме того, что первые 140 из них - числовые, а остальные - категориальные. Также, присутствует много пропущенных значений и сильный дисбаланс классов. Необходимо определиться, какую метрику качества использовать при построении классификатора, а после создать экономическую модель, демонстрирующую эффект от проведения маркетинговой кампании по удержанию пользователей. 

Сопутствующее соревнование на Kaggle: https://www.kaggle.com/c/telecom-clients-prediction2
